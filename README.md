# service
前台服务

### 1. 简介
个人项目,主要用于个人笔记使用.

配合 [service-web](https://github.com/li88qq/service-web) 使用.

### 2. 开发相关
1. 整体框架:Springboot 2.5.0
2. 数据库:mysql 8.0
3. 持久层框架: [fastquery](https://github.com/xixifeng/fastquery)

### 3. 功能
1. 常用开发辅助工具.包括json转换,时间戳转换,二维码显示,自定义导航等.
2. 笔记.使用markdown记录文章.
3. 登录记录和操作记录查询管理.
