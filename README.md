# 个人服务平台

## 1. 简介
- 该项目为个人项目, 主要用于搭建一个提供日常工作中使用到的工具, 一个可以保存个人文章, 查询一些技术文档信息的平台.
- 该项目当前为2.0整合版本

## 2. 应用框架
- springboot
- redis
- mysql
- mybatis
